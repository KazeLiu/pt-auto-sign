# ✨ PT 签到小助手 (PT Auto Signer)

让您告别每天繁琐的 PT 签到操作，一键轻松搞定！

## 简介

本扩展程序旨在为广大的 PT 玩家提供一个高效、便捷的签到解决方案。您只需在概览页面配置好您的站点，即可实现一键签到，自动处理各个站点的签到流程，为您节省宝贵的时间！

## 特性一览

### ⚡️ 一键完成所有签到 (附赠“全屏闪烁”特效)
在概览页点击按钮，所有站点将自动在前台依次打开并迅速关闭完成签到。请放心，这是我们为您准备的**“今日签到完成进度条”的增强视觉版！
```
在签到进行时，请勿惊慌，保持冷静，让出您的鼠标控制权 60 秒钟，它会为您带来积分回报！😉 (未来版本会尝试在某些风控弱的网站优化为后台静默签到)
```
### 🛠️ 模块化适配
针对不同 PT 站点的类型（如 NexusPHP、海胆、幼儿园等）设计了独立的签到策略，确保高成功率。

### ✅ 状态清晰可见
实时显示每个站点的签到状态，今日是否已签到，让您一目了然。

📦 安装与使用

安装扩展程序：

    Chrome应用商店安装（暂时还没有发布）

打开设置页面：

    点击浏览器工具栏上的扩展程序图标，即可打开概览/设置页。

开始签到：

    在概览页面中，点击 "一键全部签到" 按钮即可。
    您也可以点击列表中的 "再次尝试签到" 按钮，单独对某个站点进行操作。

## 🔧 技术栈

Vue 3 + JavaScript (ES6+)。

WebExtension API (chrome.tabs, chrome.scripting, browser.storage)。

Element Plus (UI 框架)。
